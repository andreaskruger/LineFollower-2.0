This folder contains files related to the "Path Navigation" video of the Mobile Robotics training

examples - Files used in the video
[~] vexPathNavigationMap.png : Example image of a robot course/field that has the full course for predefined path navigation
[~] pathNavigation_Logic_start.slx : Example starter model to design predefined path navigation
[~] segmentA_logic.slx : Example model that simulates Segment A logic only 
[~] segmentB_logic.slx : Example model that simulates Segment B logic only 
[~] segmentC_logic.slx : Example model that simulates Segment C logic only 
[~] segmentD_logic.slx : Example model that simulates Segment D logic only
[~] pathNavigation_Logic_Populated.slx : Example model that populates the the on-off variation of all the segments
[~] pathNavigation_Logic_soln.slx : Example model that shows predefined path navigation implementing the on-off variation of all the algorithms
[~] pathNavigation_PID_start.slx : Example starter model to design predefined path navigation including PID controllers
[~] DistCtrl_PI.slx : Example model that simulates distance dead reckoning using proportional-integral (PI) control
[~] HeadCtrl_PI.slx : Example model that simulates heading dead reckoning using proportional-integral (PI) control
[~] pathNavigation_PID_soln.slx : Example model that shows predefined path navigation and also shows how to call PID controller blocks from Stateflow using Simulink functions
[~] pathNavigation_Logic_VEX.slx : Example model deploys predefined path navigation implementing the on-off variation of all the algorithms to the VEX EDR robot
[~] pathNavigation_PID_VEX.slx : Example model deploys predefined path navigation using Stateflow with PID controller blocks to the VEX EDR robot
[~] robotParameters.m : Example MATLAB code file defining robot parameters. This file is specified to be called in the PreLoadFcn model callback of the Simulink models
[~] wheelMotorLUT.mat : Example lookup table data for the VEX EDR motors
[~] wheelMotorModel.mat : Example motor model for the VEX EDR motors
[~] examplePathNavigationMap.mat : Example obstacle map generated from vexPathNavigationMap.png and exported using the Simulation Map Generator app

exercises - Files for exercise and practice tasks
[~] PathNavigation_Exercise.pdf : Introduces the exercise tasks and presents respective solutions
[~] pathNavigation_customMap_start.slx : Starter model for custom path navigation exercise
[~] customTrackMap.mat : Map data file for the example custom path's image 
[~] customTrack.jpg : Example of a custom path's image
