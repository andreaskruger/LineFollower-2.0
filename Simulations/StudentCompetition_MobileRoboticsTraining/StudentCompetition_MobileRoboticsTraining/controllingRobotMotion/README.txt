This folder contains files related to the "Controlling Robot Motion" video of the Mobile Robotics training

examples - Files used in the video
[~] odometer1Wheel_soln.slx : Example model for calculating distance traveled by 1 wheel using Encoder sensors
[~] odometer2Wheels_soln.slx : Example model for calculating distance traveled by a 2 wheeled differential drive robot using Encoder sensors
[~] deadReckoningSF_soln.slx : Example model that shows dead reckoning logic in Stateflow
[~] deadReckoningSim_soln.slx : Example model that simulates dead reckoning logic in Stateflow using the Mobile Robotics Training library
[~] deadReckoning_VEX.slx : Example model deploys dead reckoning logic to the VEX EDR robot
[~] robotParameters.m : Example MATLAB code file defining robot parameters. This file is specified to be called in the PreLoadFcn model callback of the Simulink models
[~] wheelMotorLUT.mat : Example lookup table data for the VEX EDR motors

exercises - Files for exercise and practice tasks
[~] ControllingRobotMotion_Exercise.pdf : Introduces the exercise tasks and presents respective solutions
[~] deadReckoning_heading_start : Starter model for heading control exercise
[~] deadReckoning_heading_solution.slx : Solution model for heading control exercise
[~] deadReckoning_heading_VEX.slx : Deploy-to-VEX model for heading control exercise
