This folder contains files related to the "Using PID Controllers" video of the Mobile Robotics training

examples - Files used in the video
[~] deadReckoningPlant.slx : Example model that simulates dead reckoning logic in Stateflow including the plant motor dynamics using the Mobile Robotics Training library
[~] motorDistCtrl_start.slx : Example starter model to design distance PID control for motors
[~] motorDistCtrl_soln.slx : Example model that shows distance PID control for motors
[~] deadReckoningDistCtrl_soln.slx : Example model that simulates distance PID control for motors using the Mobile Robotics Training library
[~] deadReckoning_SF_VEX.slx : Example model deploys dead reckoning on-off logic to the VEX EDR robot
[~] deadReckoning_P_VEX.slx : Example model deploys proportional distance control (P) to the VEX EDR robot
[~] deadReckoning_PI_VEX.slx : Example model deploys proportional-integral distance control (PI) to the VEX EDR robot
[~] robotParameters.m : Example MATLAB code file defining robot parameters. This file is specified to be called in the PreLoadFcn model callback of the Simulink models
[~] wheelMotorLUT.mat : Example lookup table data for the VEX EDR motors
[~] wheelMotorModel.mat : Example motor model for the VEX EDR motors

exercises - Files for exercise and practice tasks
[~] UsingPID_Exercise.pdf : Introduces the exercise tasks and presents respective solutions
[~] deadReckoning_headingPI_start.slx : Starter model for heading control exercise
[~] deadReckoning_headingPI_solution.slx : Solution model for heading control using PID exercise
[~] deadReckoning_headingPI_VEX.slx : Deploy-to-VEX model for heading control using PID exercise
