This folder contains files related to the "Obstacle Detection Algorithms" video of the Mobile Robotics training

examples - Files used in the video
[~] vexMap.jpg : Example image of a robot course/field that has the obstacles to detect
[~] ultrasonicSensor_VEX.slx : Example model to explore the ultrasonic sensor of a VEX EDR robot through 'External Mode'
[~] obstacleDetection_start.slx : Example starter model to design obstacle detection algorithms
[~] obstacleDetection_onoff_soln.slx : Example model that shows the on-off obstacle detection algorithm
[~] obstacleDetection_PID_soln.slx : Example model that shows the proportional-integral (PI) obstacle detection algorithm
[~] obstacleDetection_onoff_VEX.slx : Example model deploys on-off obstacle detection algorithm to the VEX EDR robot
[~] obstacleDetection_PID_VEX.slx : Example model deploys proportional-integral (PI) obstacle detection algorithm to the VEX EDR robot
[~] robotParameters.m : Example MATLAB code file defining robot parameters. This file is specified to be called in the PreLoadFcn model callback of the Simulink models
[~] wheelMotorLUT.mat : Example lookup table data for the VEX EDR motors
[~] wheelMotorModel.mat : Example motor model for the VEX EDR motors
[~] exampleObstacleMap.mat : Example obstacle map generated from vexMap.jpg and exported using the Simulation Map Generator app

exercises - Files for exercise and practice tasks
[~] ObstacleAvoidance_Exercise.pdf : Introduces the exercise tasks and presents respective solutions
[~] obstacleAvoidance_uTurn_start.slx : Starter model for obstacle avoidance exercise
[~] obstacleAvoidance_uTurn_solution.slx : Solution model for obstacle avoidance exercise
[~] obstacleAvoidance_uTurn_VEX.slx : Deploy-to-VEX model for obstacle avoidance exercise
